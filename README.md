# TowerOfHanoiProject

Tower of Hanoi is a game where the player moves rings between poles.

On how to build/run the command prompt version of this game, you first download the towerOfHanoi.cpp file and run it in an IDE of your choosing.

Rules:
The Tower of Hanoi game consists of three poles and a set of rings of different sizes. All rings start on the first pole, stacked from largest at the bottom to smallest at the top. The goal is to move all rings to the third pole.

Only one ring can be moved at a time. A ring can only be placed on an empty pole or on top of a larger ring. Placing a larger ring on a smaller ring is not allowed. The player continues making moves until all rings are successfully moved to the third pole.

User input is entered as two digits. The first digit represents the source pole and the second digit represents the destination pole. For example, entering 13 means moving a ring from pole 1 to pole 3.
